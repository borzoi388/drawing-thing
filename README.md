# DrawingThing
DrawingThing is a simple pixel art thingy with some basic drawing app functions
### Features:
- Canvas of up to 100px by 100px (i know it's small)
- Saving works as pngs
- Layers so you can do cool stuff
- Fill bucket, eraser, pen with changeable size
- Undo mechanic that is able to undo layer creation and deletion as well as pen strokes

### How to run:
- Only dmg for Mac at the moment, just download and drag and drop
