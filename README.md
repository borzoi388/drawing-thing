# DrawingThing
DrawingThing is a simple pixel art thingy with some basic drawing app functions
### Features:
- Canvas of up to 100px by 100px (i know it's small)
- Saving works as pngs
- Layers so you can do cool stuff
- Fill bucket, eraser, pen with changeable size
- Undo mechanic that is able to undo layer creation and deletion as well as pen strokes

### Process:
Made in Java!
Yeah so I coded a lot of the things I didn't know were available in the java library myself :,D Including connecting lines, stroke thickness, and displaying the pixels
<br>
Overall I wanted to focus more on coding the graphics stuff I mentioned above so the UI doesn't look great, but i did learn a lot about jswing
